# APK Notes

The APK is part of the same general idea as this repository: convert or package
mind map content into an HTML-based reading experience on mobile.

Important limits:

- The APK is intended for reading, not editing.
- It may contain bugs or incomplete behavior.
- Its maintainability may be limited because it was produced entirely or largely with Codex assistance.
- The APK should be distributed as a release artifact rather than committed into repository history.

When publishing later, include:

- version label
- SHA-256 checksum
- a clear non-official disclaimer
