# Skill Overview

The primary public artifact in this project is the `mindmap-html-authoring` Codex skill.

It defines a lightweight authoring workflow:

1. Write a JSON tree.
2. Build a standalone HTML viewer.
3. Preserve the `IMG` and `RAW_TREE` contract for recovery and compatibility.

The public repository should treat the skill as the core product surface. The conversion tools are companion utilities around that skill.
