# Codex Mindmap HTML Toolkit

[中文说明](./README.zh-CN.md)

This staging copy is a public-release candidate for a **Codex skill project**.

Its main asset is `skills/mindmap-html-authoring`, a Codex skill for generating standalone interactive mind map HTML files from structured JSON input. The bundled tools extend that workflow with:

- `tools/xmind-to-html`: convert `.xmind` files into standalone HTML viewers
- `tools/html-to-xmind`: recover `.xmind` files from compatible HTML generated by this skill or the matching viewer contract

Important boundaries:

- This is not an official XMind project.
- It is not affiliated with, sponsored by, or endorsed by Xmind / Supermind Pte. Limited.
- `XMind` / `Xmind` are referenced only to describe compatibility targets.
- This repository does not implement the full XMind feature set.
- It is a viewer/generator/recovery workflow, not a full editor.
- The code was created or heavily assisted by Codex and may contain bugs or maintainability issues.
- The current viewer and sizing heuristics were originally tuned for Chinese-language content, so English-only maps may not render as neatly in all cases.
