# Codex Mindmap HTML Toolkit

[中文说明](./README.zh-CN.md)

This staging copy is a public-release candidate for a **Codex skill project**.

Its main asset is `skills/mindmap-html-authoring`, a Codex skill for generating standalone interactive mind map HTML files from structured JSON input. The bundled tools extend that workflow with:

- `tools/xmind-to-html`: convert `.xmind` files into standalone HTML viewers
- `tools/html-to-xmind`: recover `.xmind` files from compatible HTML generated by this skill or the matching viewer contract

## Ideal Workflow

In the author's view, the most natural way to use this skill is as a
natural-language workflow with Codex:

1. Describe the mind map you want in plain language.
2. Let Codex generate or revise a customized, usable mind map HTML file.
3. Convert that HTML into an `.xmind` file when manual editing is needed.
4. Read the result on Android devices through the companion APK when mobile viewing is useful.

The intended experience is not just file conversion. It is a loop between
natural-language guidance, Codex-assisted generation, optional manual editing
in XMind, and lightweight mobile reading.

## HTML Controls

- Left mouse drag: pan normally when dragging on the background.
- Middle mouse drag: pan from anywhere, including over images, text, and branches.
- Mouse wheel: zoom in and out.
- Click a branch node: expand or collapse that branch.
